Build Manual Drag-and-Drop Program Builder + Exercise Intelligence Layer Phase 1 for TrainEfficiency.

GOAL:
Create a professional visual workout programming system that combines:
- TrainChat AI generation
- manual coach editing
- drag-and-drop program construction
- reusable templates
- exercise intelligence/media

This should NOT replace TrainChat.
It should become the visual execution/editor layer on top of TrainChat.

====================================================
1. CORE CONCEPT
====================================================

Current:
TrainChat generates programs.

Next:
Coaches can:
- visually edit programs
- drag/drop exercises
- reorder weeks/sessions
- create supersets/circuits
- attach videos/cues
- save templates
- combine AI + manual programming

====================================================
3. VISUAL PROGRAM BUILDER
====================================================

Create:
 /org/:slug/programs/:programSlug/builder

Coach builder should support:

A. Week View
- drag/drop weeks
- duplicate week
- rename week
- reorder sessions

B. Session View
- drag/drop exercises
- reorder exercises
- duplicate exercise
- delete exercise
- inline editing

C. Exercise Cards
Each exercise card shows:
- name
- sets/reps/load
- RPE
- notes
- media thumbnail
- tags

D. Grouping
Allow:
- supersets
- circuits
- giant sets
- contrast pairings

Visual grouping containers.
