TrainEfficiency Revenue Attribution Audit

Agent → Message → Outcome → Revenue Chain Verification

Objective:

The communications system has passed governance, safety, autonomy, and learning audits.

The next validation is business attribution.

We need to prove that TrainEfficiency can accurately trace revenue back to:

* Agent
* Workflow
* Domain
* Message
* Learning Rule
* Human Approval Decision

and ultimately determine:

What generated revenue and why?

Phase 3 — Reply Attribution Audit

For every outcome with:

status = 'replied'

Verify:

* originating message exists
* domain exists
* action exists
* workflow exists (if applicable)

Calculate:

Reply Attribution Completeness %

Formula:

Replies with valid parent action ÷ total replies

Phase 4 — Booking Attribution Audit
