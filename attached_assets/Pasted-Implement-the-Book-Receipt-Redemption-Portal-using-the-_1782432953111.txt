* leadId (nullable)
* email
* receiptFileUrl

## API

Create:

POST /api/book-funnel/receipt

Behavior:

Validate:

* Email
* Supported file type
* File size

Attempt to associate the receipt with an existing lead by email.

Store uploaded receipt using the application's existing upload/storage approach.

Create submission.

Create event:

book_receipt_uploaded

Return success.
