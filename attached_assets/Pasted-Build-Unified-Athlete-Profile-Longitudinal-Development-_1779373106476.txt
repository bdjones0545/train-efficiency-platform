Build Unified Athlete Profile + Longitudinal Development Graph Phase 1 for TrainEfficiency.

GOAL:
Create a single “Athlete Intelligence Profile” that combines all athlete systems into one longitudinal development view.

This becomes:
- the athlete’s operating profile
- the coach’s intelligence dashboard
- the foundation for recruiting and long-term development

====================================================
1. CORE CONCEPT
====================================================

Every athlete should have:

Unified Athlete Profile

Combining:
- PR history
- workout adherence
- readiness trends
- fatigue trends
- education completion
- intervention history
- streaks
- attendance
- notes
- badges
- timeline events
- body metrics (future-ready)
- recruiting data (future-ready)

2. NEW PAGE
