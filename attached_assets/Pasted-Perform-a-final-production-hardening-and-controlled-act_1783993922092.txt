```text

```

The following route must not use browser-session ADMIN authorization as its primary protection:

```text

```

```text

```

Verify:

* missing token returns 401
* incorrect token returns 401
* valid token reaches payload validation
* a valid ADMIN browser session without the internal token is rejected
* a valid internal token without a browser session is accepted
* token comparison is timing-safe
* token is never logged
* token is never included in audit payloads
* token is never exposed to the client bundle
* token validation fails closed when the environment variable is missing or shorter than the required minimum

Correct any misleading comments, tests, route descriptions, or documentation that describe the internal endpoint as ADMIN-only.
