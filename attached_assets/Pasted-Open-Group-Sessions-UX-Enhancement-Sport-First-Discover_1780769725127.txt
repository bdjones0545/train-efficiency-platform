Open Group Sessions UX Enhancement — Sport-First Discovery Model

Objective:
Redesign Open Group Sessions around how athletes and parents actually search for training opportunities. Preserve all existing APIs, functionality, permissions, booking logic, filters, registration flows, waitlists, and session management. This is a UX/UI enhancement only.

PHASE 8 — INFORMATION PRIORITY RULE

Every session card must allow a parent to answer these questions within 2 seconds:

✓ What sport is this?
✓ Where is it?
✓ Who coaches it?
✓ What level is it?
✓ What age group is it for?
✓ How many spots are left?
✓ Can my child join?

If any of these require opening the session details, the card hierarchy still needs improvement.
