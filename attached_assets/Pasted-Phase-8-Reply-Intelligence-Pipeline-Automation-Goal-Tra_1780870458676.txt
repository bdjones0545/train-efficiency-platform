Phase 8 — Reply Intelligence & Pipeline Automation

Goal:
Transform Opportunity Acquisition from an outbound outreach system into a two-way conversation intelligence system.

Current State:

Discovery
→ Qualification
→ Outreach Draft
→ Approval
→ Send
→ Track

New State:

Discovery
→ Qualification
→ Outreach Draft
→ Approval
→ Send
→ Reply Detection
→ Reply Intelligence
→ Pipeline Automation
→ Follow-Up Recommendation

Important:

Do NOT enable autonomous reply sending.

Do NOT enable autonomous negotiation.

Do NOT enable autonomous contract discussions.

This phase is intelligence, classification, and workflow automation only.

API ENDPOINTS

Returns reply events newest first.

GET /api/opportunity-acquisition/replies/:id

Returns full reply detail.

POST /api/opportunity-acquisition/replies/:id/generate-followup

Generates follow-up draft.

POST /api/opportunity-acquisition/replies/reprocess/:id

Re-runs classification.
