Upgrade the TrainEfficiency Education AI generation system to maximize educational quality, coaching realism, scientific grounding, and athlete engagement.

Current system already supports:

* One-shot full pathway generation
* Modules
* Per-module quizzes
* Final pathway assessment
* Video embedding
* Sequential progression
* Badge awards

GOAL:
Transform the Education Builder from “AI-generated content” into an elite adaptive athlete learning engine that feels authored by a real high-level performance coach.

Implement the following upgrades:

1. CONTENT QUALITY SYSTEM PROMPT LAYER

Create a centralized education-generation system prompt used for ALL pathway/module/quiz generation.

The AI should:

* Teach like an elite sports performance coach
* Use concise athlete-friendly language
* Avoid generic AI phrasing and fluff
* Avoid repetitive sentence structures
* Avoid textbook-style explanations
* Prioritize actionable coaching
* Connect concepts directly to performance outcomes
* Use real-world sports examples
* Reinforce practical application
* Maintain evidence-based recommendations
* Adapt explanations to athlete age level
* Use encouraging but professional tone

The AI should NEVER:

* Overuse motivational filler
* Use vague wellness language
* Recommend unsafe practices
* Promote pseudoscience
* Recommend supplements inappropriately for youth athletes
* Generate excessively long paragraphs

2. COACH PHILOSOPHY PROFILES

Add organization-level or coach-level generation preferences.

Examples:

* teaching style
* coaching philosophy
* terminology preferences
* preferred tone
* banned terminology
* emphasis areas

Example settings:

* “Use athlete-friendly language”
* “Emphasize recovery and long-term development”
* “Avoid bodybuilding terminology”
* “Focus on movement quality and readiness”

These preferences should automatically influence all AI-generated education.

3. READING LEVEL ADAPTATION

Middle school:

* shorter sections
* simpler wording
* more examples

High school:

* moderate detail
* practical sports examples

College/pro:

* deeper physiology explanations
* advanced recovery/performance concepts

The AI should automatically adapt:

* reading complexity
* lesson length
* terminology
* examples
* quiz difficulty

4. KEY TAKEAWAYS SYSTEM

Every module should auto-generate:

* 3–5 key takeaways
* quick recap cards

These should appear:

* at the end of lessons
