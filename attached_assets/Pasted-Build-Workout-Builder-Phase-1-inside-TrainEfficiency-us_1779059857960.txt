Build Workout Builder Phase 1 inside TrainEfficiency using the existing TrainChat API integration.

IMPORTANT CONTEXT:
TrainChat is already connected through the TrainChat Integration system.
Do NOT rebuild TrainChat.
Do NOT recreate programming logic inside TrainEfficiency.

TrainEfficiency should:
- orchestrate workflows
- collect athlete/team context
- send generation requests to TrainChat
- store returned programs
- assign programs
- display workouts in the athlete portal
- track completion/progress

TrainChat remains:
- the AI programming engine
- periodization engine
- exercise reasoning engine
- refinement engine

All generation must flow:
TrainEfficiency frontend
→ TrainEfficiency backend
→ TrainChat API
→ TrainEfficiency storage/display layer

====================================================
3. BACKEND ENDPOINTS
====================================================

Create:

GET /api/org/workout-builder/bootstrap

Returns:
- org info
- current user/membership
- teams
- athletes
- existing programs
- TrainChat integration status

POST /api/org/workout-builder/generate

Purpose:
Generate a program through TrainChat.

Input:
- targetType
  (team or athlete)
- athleteUserIds
- teamId
- goal
- sport
- durationWeeks
- daysPerWeek
- equipment
- constraints
- coachNotes

Backend responsibilities:
- validate org permissions
- verify TrainChat integration exists
- gather athlete/team context
- gather PR summaries
- gather coach notes summaries
- gather recent training activity
- package context
- call trainChatClient.generateProgram()
- store returned program
- parse/store sessions
- return formatted program

POST /api/org/workout-builder/:programId/assign

Assign program to:
- athlete
or
- team

GET /api/org/workout-builder/programs/:programId

Return:
- program
- sessions
- assignments
- generated summary
- TrainChat rationale

PATCH /api/org/workout-builder/programs/:programId

Allow:
- title edits
- description edits
- status updates

POST /api/org/workout-builder/programs/:programId/edit

Purpose:
Send refinement instruction to TrainChat API.

Examples:
- reduce lower body volume
- increase speed emphasis
- shorten sessions
- modify for knee pain

Update stored program afterward.

POST /api/org/workout-builder/sessions/:sessionId/complete

Athlete marks workout complete.
