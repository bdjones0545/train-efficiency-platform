Build AI Coach Content Builder for Education Modules in TrainEfficiency.

GOAL:
Allow coaches/orgs to create, edit, assign, and track custom education pathways using AI assistance.

This should expand the existing Nutrition Education module into a reusable Education Builder system.

====================================================
1. CORE CONCEPT
====================================================

Coaches can create education pathways for:
- Nutrition
- Recovery
- Hydration
- Sleep
- Mindset
- Team Standards
- Injury Prevention
- Recruiting Education
- Custom topics

AI should help generate:
- lesson content
- key takeaways
- quizzes
- module summaries
- athlete-friendly rewrites
- coach-approved edits

IMPORTANT:
AI-generated content must always be coach-reviewed before publishing.

====================================================
2. ROUTES
====================================================

Create coach builder route:

/org/:slug/coach/education-builder

Create athlete education route:

/org/:slug/education/:pathwaySlug

Keep existing nutrition route working:
- /org/:slug/nutrition can redirect to nutrition pathway
or remain as shortcut.

====================================================
4. AI BUILDER FEATURES
====================================================

Inside Education Builder add:

A. Generate Pathway with AI
Coach enters:
- topic
- athlete age group
- sport/team
- tone
- number of modules
- difficulty level
- goal

AI returns:
- pathway title
- module outline
- learning goals
- suggested quiz structure

Coach can:
- accept
- edit
- regenerate
- discard

B. Generate Module Content
For each module AI can generate:
- lesson sections
- simple athlete language
- key takeaways
- examples
- short summary

C. Generate Quiz
AI creates:
- 3–5 questions
- multiple choice answers
- correct answer
- explanation

D. Rewrite Tools
Buttons:
- Make simpler
- Make more athlete-friendly
- Make shorter
- Make more professional
- Add examples
- Turn into bullet points

====================================================
5. SAFETY RULES FOR AI CONTENT
====================================================

AI should avoid:
- medical advice
- supplement prescriptions
- calorie/macronutrient prescriptions
- eating disorder language
- body-shaming
- diagnosis/treatment claims
- extreme dieting advice

For nutrition content:
- use education-only language
- performance fueling
- hydration
- recovery basics
- refer to qualified professionals for individual plans

All AI content must show:
“AI draft — coach review required.”
