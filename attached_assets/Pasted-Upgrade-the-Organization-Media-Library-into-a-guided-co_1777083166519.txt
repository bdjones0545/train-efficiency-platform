Upgrade the Organization Media Library into a guided, conversion-optimized media builder with live preview, completion tracking, and smarter UX.

Goal:
Transform the current “upload files” UI into a structured system that helps coaches build a high-converting landing page.

========================
PHASE 1 — GUIDED BUILD FLOW
========================

Replace the empty state with a 3-step guided system.

Add a new component at the top:

MediaBuildProgress

Steps:
1. Hero Media (required)
2. Training Showcase (recommended)
3. Facility Media (recommended)

UI:
- Progress bar (0–100%)
- Each step shows:
  - status: complete / incomplete
  - short description
- Example:
  [ ] Hero Media — Add a video or image that represents your training
  [ ] Training Showcase — Show how you coach (3–5 clips recommended)
  [ ] Facility — Show your environment and equipment

Logic:
- Hero complete = at least 1 active hero item
- Training complete = at least 3 active training items
- Facility complete = at least 2 active facility items

Display:
- Show % complete
- Show green checkmarks as sections are completed

========================
PHASE 2 — LIVE LANDING PAGE PREVIEW
========================

Add a toggle at top right:

[ Preview Page ]

Behavior:
- Opens a side panel (desktop) or full-screen modal (mobile)
- Renders actual org landing page using current media

Implementation:
- Reuse public landing page component
- Inject current org media state (no refetch required if cached)

Live updates:
- After upload, reorder, or toggle active:
  → preview updates immediately

Mobile:
- Use full-screen overlay (Sheet)
- Add close button top-right
