Build Communication Automation + Multi-Channel Outreach Engine Phase 1 for TrainEfficiency.

GOAL:
Create an intelligent communication layer that automates coach outreach, athlete accountability, reminders, encouragement, and intervention follow-up.

The system should connect:
- Command Center
- Risk Engine
- Education
- Workout Builder
- Notifications
- Adaptive Workflows

====================================================
1. CORE CONCEPT
====================================================

The platform should intelligently communicate with:
- athletes
- team coaches
- guardians
- org coaches

Examples:

Low readiness:
→ supportive athlete reminder
→ coach alert

Missed workouts:
→ accountability reminder
→ follow-up escalation

Education overdue:
→ reminder + encouragement

PR achieved:
→ celebration message

Streak milestone:
→ motivational message

====================================================
2. COMMUNICATION CHANNELS
====================================================

Support:
- in-app notifications
- email
- SMS (future-ready)
- guardian notifications
- coach inbox

Delivery preferences per user.

====================================================
4. AUTOMATED MESSAGE TYPES
====================================================

Seed templates for:

- missed workout reminder
- low readiness encouragement
- hydration reminder
- education overdue
- PR celebration
- streak milestone
- coach follow-up request
- intervention reminder
- upcoming session reminder
- recovery encouragement

Tone:
supportive, performance-focused, positive.

====================================================
5. AI MESSAGE GENERATION
====================================================

Create:
POST /api/org/communication/ai-generate

AI receives:
- athlete context
- risk data
- streaks
- recent activity
- intervention context

AI generates:
- athlete-facing message
- coach-facing summary
- guardian summary (optional)

Rules:
- encouraging tone
- no medical claims
- no alarming language
