- any route/service that calls sendAgentEmail() or replyFromAgentInbox()

Required:
Create a single centralized guard before any AgentMail outbound send:
- check org AI governance / automation settings
- check draft-only mode
- check approvalRequired
- check risk level

- write outbound_email_audit_log with status='blocked' or 'requires_approval'

- write outbound_email_audit_log
- write agent_operating_timeline
