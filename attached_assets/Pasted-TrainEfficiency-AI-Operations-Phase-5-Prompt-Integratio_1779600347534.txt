PART 2 — Integration Runtime Layer

Create:
server/integration-runtime.ts

Responsibilities:

* central runtime for ALL external integrations
* enforce governance before external execution
* handle retries/failures/rate limits
* standardize external API responses
* normalize provider errors
* attach audit logging
* enforce integration health checks
* route execution through workflow job queue

Functions:

* executeIntegrationAction()
* validateIntegrationHealth()
* refreshIntegrationState()
* classifyProviderError()
* normalizeProviderResponse()
* checkIntegrationGovernance()
* pauseIntegration()
* resumeIntegration()

NO external action should bypass this runtime.
