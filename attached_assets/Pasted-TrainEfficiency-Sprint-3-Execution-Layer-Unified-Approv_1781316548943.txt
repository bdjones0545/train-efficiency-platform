Phase 1 — Unified Approval Center

Create:

/admin/action-center

* Hermes recommendations
* Gmail draft approvals
* Autonomous queue items
* Scheduling assistant approvals

* Action title
* Source agent

Phase 4 — Cross-Agent Coordination Layer

Create:

server/services/cross-agent-coordination-service.ts

Before creating any action:

Check:

* gmail_thread_id
* source_conversation_id
* prospect_id
* lead_id
* action_type

If duplicate exists:

Do not create new action.

Instead:

* merge
* attach source references
* increase support score

Store:

coordination_decisions
