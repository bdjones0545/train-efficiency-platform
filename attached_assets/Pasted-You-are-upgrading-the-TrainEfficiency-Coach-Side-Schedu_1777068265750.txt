========================
PHASE 1 — CLIENT BOOKINGS LOOKUP (CRITICAL)
========================

Problem:
Agent cannot retrieve a specific client’s upcoming sessions.
Blocks flows like:
- “Make David’s session recurring”
- “When is Sarah’s next session?”
- “Reschedule John’s next booking”

Solution:
Create new tool: get_client_bookings

Inputs:
- clientId
- optional date range (default: next 30 days)

Returns:
- list of upcoming bookings:
  - bookingId
  - date
  - startTime
  - endTime
  - coachId
  - serviceId
  - location (if available)

Agent behavior:
- Must use this tool anytime a prompt references “their session”, “next session”, or “existing booking”
- Must NOT rely on get_org_schedule for client-specific queries
